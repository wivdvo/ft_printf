# ft_printf

42 school project for which I created my own implementation of printf.

## Installation
git clone git@github.com:wivdvo/ft_printf.git ft_printf  
cd ft_printf  
make  

## Usage
Use like original printf.  
Supports:
- %c (chars)
- %s (strings)
- %d (integers)
- %i (integers)
- %u (unsigned int)
- %x (lowercase hex)
- %X (uppercase hex)
- %p (pointer adress)
- %% (prints '%')
